# Aplicação desenvolvida para o Projeto de prever possíveis resultados de jogos da copa do mundo de 2022.

Este repositório contém a Aplicação de Machine Learning construída com o framework Streamlit do Python para o Projeto de prever possíveis resultados e seleção vencedora da copa do mundo de 2022 (https://github.com/luislauriano/Data_Science/tree/master/Possiveis%20resultados%20copa%20do%20mundo%202022). Após a aplicação ter sido finalizada, realizei o deploy com o heroku.
